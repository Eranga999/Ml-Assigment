# Sri Lankan Hotel Resource Management & Booking Analysis 🏨⚡

## Project Overview
This project addresses real-world resource management challenges in the Sri Lankan hospitality sector. By utilizing machine learning, we analyze historical booking data to predict cancellations and optimize energy/resource allocation. This allows hotel managers to proactively adjust staffing and inventory based on predicted occupancy.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Environment:** Jupyter Notebook (Anaconda)
* **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn

## 📊 Dataset Information
* **Source:** [Hotel Booking Demand Dataset (Kaggle)](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand)
* **Size:** 119,390 observations
* **Attributes:** 32 variables including lead time, hotel type, and arrival dates.
* **Context:** Adapted for the Sri Lankan tourism industry to handle seasonal demand peaks.

## 🧠 Machine Learning Algorithms
To meet assignment requirements for a team of four, we implemented and compared four distinct supervised learning algorithms:
1. **Logistic Regression** (Baseline model)
2. **K-Nearest Neighbors (KNN)**
3. **Decision Tree**
4. **Random Forest** (High-accuracy ensemble)
